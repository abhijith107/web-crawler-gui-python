# Web Crawler with GUI

This is a Python application that allows you to crawl a website, fetch network information, and search for specific keywords in the crawled data. The GUI is built using Tkinter.

## Installation

1. Clone this repository to your local machine:


2. Navigate to the project directory:


3. Install the required dependencies using pip:


## Usage

1. Run the `crawl.py` script:


2. Enter the starting URL in the provided input field and click on "Start Crawling".
3. You can search for specific keywords in the crawled data by entering the keyword in the respective input field and clicking on the "Search" button.
4. The results will be displayed in the output area. You can also display all the crawled data by clicking on the "Display Crawled Data" button.

## Dependencies

- fuzzywuzzy
- requests
- scrapy
- selenium
